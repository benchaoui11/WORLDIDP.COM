1968 Vienna Convention on Road Traffic

Entity page explaining the 1968 Vienna road-traffic convention and why national IDP implementation can still vary.

Direct Answer

The convention framework helps define international driving-permit formats, but it does not make WorldIDP a government issuing authority and it does not override national implementation rules.

Explainer

What it governs

  • Updates the road-traffic framework for many participating countries.
  • Some destinations reference 1968 Vienna convention IDPs or licences.
  • A 1968 framework does not guarantee digital-only acceptance.
  • WorldIDP is a private translation-support service, not a treaty authority.
